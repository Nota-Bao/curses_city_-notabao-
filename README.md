# curses_city
 Basic Sim City style game written in C++ using ncurses for ascii graphics 

 See http://cbabuska.com/CursesCity

 # Maintained version:
 This version is now being maintained by Nota Bao, adding small changes to expand upon the game. First change was fix the game not compiling due to an error to the date object, as well as changing the main movement from HJKL to ARROW-KEYS.

 ## Installation:
 pull -> make -> ./Curses_City

 
 ## Instructions and Keybinds:
 'r' : New Residential Zone
 
 'c' : New Commercial Zone
 
 'i' : New Industrial Zone
 
 'o' : New Road
 
 'd' : New Rail Road
 
 'p' : New Police Station
 
 'f' : New Fire Station
 
 's' : New School
 
 'm' : New Hospital
 
 'a' : New Park
 
 'w' : New Coal Power Plant (Cheap)
 
 'n' : New Nuclear Power Plant (Expensive)
 
 'x' : Bulldoze

 '1' : Slow speed
 
 '2' : Medium speed
 
 '3' : Fast speed

 'q' : Pause game
 
 'Q' : Exit game

 Movement: Arrow keys
